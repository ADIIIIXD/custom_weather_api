WEATHER DATA API

URL Format: http://127.0.0.1:5000/api/v1/station/date

for single date of single station: http://127.0.0.1:5000/api/v1/STAID/YYYY-MM-DD

for Full data of single station: http://127.0.0.1:5000/api/v1/STAID

for a specific year for a single station: http://127.0.0.1:5000/api/v1/yearly/STAID/YYYY

Below is the table consisting of all the temperature data

STAID: Station ID

STANAME = Station Name

CN: Country Code

LAT: Latitude in degrees:minutes:seconds (+: North, -: South)

LON: Longitude in degrees:minutes:seconds (+: East, -: West)

HGTH: Station elevation in meters
